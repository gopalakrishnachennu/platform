Platform Quick Access

Use these commands to open Backstage, ArgoCD, Grafana, and the dev app quickly.

Dashboard links (open after tunnels start)

Open Backstage Open ArgoCD Open Grafana Open Dev App

Start tunnels first, then these links will work.

One command (opens all tunnels)

bash scripts/open_access_tunnels.sh

This script opens separate Terminal tabs and starts all 4 port-forwards.

Start cost-safe mode (recreate and reinstall)

bash scripts/start_cost_safe_mode.sh

Recreates low-cost cluster and reinstalls IDP, ArgoCD, and observability stack.

Health check (single command PASS/FAIL)

bash scripts/health_check_all.sh

Verifies cluster, namespaces, ArgoCD apps, workloads, and alert rules.

Backstage

kubectl port-forward svc/backstage 7007:7007 -n idp

ArgoCD

kubectl port-forward svc/argocd-server 8080:443 -n gitops

Grafana

kubectl port-forward svc/kube-prometheus-grafana 3000:80 -n monitoring

Dev app test

kubectl port-forward svc/demo-app 18080:80 -n dev

Stop GCP cost (recommended when idle)

bash scripts/stop_gcp_costs.sh

This deletes the GKE cluster (`platform-lab`) to stop compute charges. You can recreate later.
