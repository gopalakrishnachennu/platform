Run Order Timeline

Follow this sequence for the smoothest daily workflow.

Step 1 Start

Run cost-safe setup to create cluster and core services.

Step 2 Open

Start all tunnels and open Backstage, ArgoCD, Grafana, Dev app.

Step 3 Verify

Run health checks and confirm app sync/deployments.

Step 4 Stop

Delete cluster when idle to control cost usage.

Start Cost-Safe Mode

Recreate low-cost cluster and install core stack end to end.

bash scripts/start_cost_safe_mode.sh

Open All Tunnels

Starts Backstage, ArgoCD, Grafana, and Dev app tunnels in terminal tabs.

bash scripts/open_access_tunnels.sh

Health Check

Validate namespaces, pods, ArgoCD sync, and alert rules in one run.

bash scripts/health_check_all.sh

Backstage Tunnel

Use individual tunnel command when needed.

kubectl port-forward svc/backstage 7007:7007 -n idp

ArgoCD Tunnel

Secure tunnel to ArgoCD server.

kubectl port-forward svc/argocd-server 8080:443 -n gitops

Grafana Tunnel

Open Grafana UI on local port 3000.

kubectl port-forward svc/kube-prometheus-grafana 3000:80 -n monitoring

Dev App Tunnel

Open your sample app running in `dev` namespace.

kubectl port-forward svc/demo-app 18080:80 -n dev

Stop GCP Cost (When Idle)

Deletes `platform-lab` cluster to stop compute spend. Recreate later with start-cost-safe mode.

bash scripts/stop_gcp_costs.sh
